# Security Policy

## Supported Versions
This repository is a portfolio/reference implementation. It is not a supported production service.

## Reporting a Vulnerability
If you discover a security issue in these examples, open a GitHub issue with:
- a clear description
- steps to reproduce
- suggested remediation

## Scanning Guidance
This repo includes optional scanning examples:
- checkov (IaC scanning)
- tfsec (IaC scanning)
- SLXP-style adversarial reports (simulated)

These are demonstration patterns. Replace with enterprise-approved tooling in production.
